# JavaScript前置知识学习清单（适配数据库课程设计）
本清单按学习顺序排列，每个模块包含「核心重点」「入门资源」「实操小案例」，聚焦“快速能用”，所有内容均服务于“数据库+前端可视化”的课程设计需求。


## 一、HTML：搭建页面骨架（建议1-2天）
### 核心重点（只学这些就够）
1. 语义化标签：`<table>`（展示数据库查询结果）、`<form>`（用户输入查询条件/提交数据）、`<button>`（触发查询/添加/修改操作）、`<div>`（划分图表/表格区域）
2. 关键属性：`id`（给元素唯一标识，如`id="dataTable"`，方便JS定位）、`class`（批量控制样式，如`class="form-item"`）
3. 表单元素：`<input type="text"`（输入查询关键词）、`<select>`（下拉选择查询条件，如“按学号/姓名查询”）

### 入门资源
- 官网教程（免费）：[W3Schools HTML教程](https://www.w3schools.com/html/)（中文，示例直观，可在线编辑）
- 速查工具：[MDN HTML元素参考](https://developer.mozilla.org/zh-CN/docs/Web/HTML/Element)（查具体标签用法）

### 实操小案例（直接对接课程设计）
**案例：学生信息查询页面结构**  
写出包含“查询表单”和“数据展示表格”的HTML文件，结构如下：
```html
<!-- 查询表单 -->
<form id="queryForm">
  <label>学号：</label>
  <input type="text" id="studentId" placeholder="输入学号">
  <button type="button" id="queryBtn">查询</button>
</form>

<!-- 数据展示表格 -->
<table id="dataTable" border="1">
  <tr>
    <th>学号</th>
    <th>姓名</th>
    <th>专业</th>
    <th>成绩</th>
  </tr>
  <!-- JS后续会动态插入数据行 -->
</table>
```


## 二、CSS：美化页面（建议1天）
### 核心重点（拒绝复杂，够用即止）
1. 基础样式：字体（`font-family`）、颜色（`color`）、边距（`margin/padding`，避免元素挤在一起）、边框（`border`，给表格/按钮加边框）

3. 常用选择器：`#id`（定位单个元素，如美化表格）、`.class`（批量美化表单元素）

### 入门资源
- 官网教程（免费）：[W3Schools CSS教程](https://www.w3schools.com/css/)（重点看“CSS Basics”和“CSS Flexbox”）
- 速查工具：[CSS常用属性速查](https://developer.mozilla.org/zh-CN/docs/Web/CSS/Reference)（只查清单里提到的属性）

## 三、JavaScript：实现交互核心（建议3-5天）
### 模块1：JS基础语法（1天）
#### 核心重点
1. 变量：用`let`（可修改）、`const`（不可修改，如固定的接口地址），避免`var`；
2. 函数：普通函数（如`function queryData() {}`，封装查询逻辑）、箭头函数（简化写法，后续用在`forEach`里）；
3. 循环与判断：`for`循环（遍历少量数据）、`forEach`（遍历数据库返回的数组数据）、`if-else`（处理查询结果为空的情况）；
4. 数据类型：重点掌握「数组」（存多条数据库数据）和「对象」（存单条数据，如`{ id: 1, name: "张三" }`）。

#### 入门资源
- 官网教程（免费）：[MDN JavaScript基础](https://developer.mozilla.org/zh-CN/docs/Learn/JavaScript/First_steps)（只看“变量”“函数”“数组”章节）
- 练习工具：[FreeCodeCamp JS基础练习](https://www.freecodecamp.org/chinese/learn/javascript-algorithms-and-data-structures/#basic-javascript)（做前10道题即可）

#### 实操小案例
**案例：定义“学生数据”数组并遍历**  
```javascript
// 模拟数据库返回的学生数据（数组+对象）
const studentData = [
  { id: "2023001", name: "张三", major: "计算机", score: 85 },
  { id: "2023002", name: "李四", major: "软件", score: 92 }

```


### 模块2：DOM操作（2-3天，核心中的核心）
#### 核心重点
1. 查找元素：`document.getElementById()`（通过id找表格/按钮，最常用）、`document.querySelector()`（按选择器找元素）；
2. 绑定事件：给按钮加`click`事件（如点击“查询”触发`queryData()`函数）；
3. 更新页面：
   - 创建元素：`document.createElement()`（创建`<tr>``<td>`，插入表格数据）；
   - 插入元素：`parent.appendChild()`（把创建的`<td>`加到`<tr>`，再把`<tr>`加到表格）；
   - 清空元素：`table.innerHTML = ""`（查询新数据前清空旧表格内容）。

#### 入门资源
- 官网教程（免费）：[MDN DOM操作教程](https://developer.mozilla.org/zh-CN/docs/Learn/JavaScript/Client-side_web_APIs/DOM_manipulation)（重点看“创建和插入元素”）

#### 实操小案例
**案例：点击按钮，将模拟数据插入表格**  
```javascript
// 1. 找到按钮和表格
const queryBtn = document.getElementById("queryBtn");
const dataTable = document.getElementById("dataTable");

// 2. 给按钮加点击事件
queryBtn.addEventListener("click", () => {
  // 模拟数据库数据
  const studentData = [
    { id: "2023001", name: "张三", major: "计算机", score: 85 },
    { id: "2023002", name: "李四", major: "软件", score: 92 }
  ];

  // 3. 清空表格原有数据（除表头）
  const tbody = dataTable.querySelector("tbody") || dataTable;
  tbody.innerHTML = "";

  // 4. 遍历数据，插入表格
  studentData.forEach(student => {
    const tr = document.createElement("tr"); // 创建行
    // 插入单元格
    tr.innerHTML = `
      <td>${student.id}</td>
      <td>${student.name}</td>
      <td>${student.major}</td>
      <td>${student.score}</td>
    `;
    tbody.appendChild(tr); // 把行加到表格
  });
});
```


## 四、前后端交互：Fetch + JSON（建议1天）
### 核心重点
1. Fetch 基础：发送GET请求（查询数据）、POST请求（提交新数据），核心流程：
   - `fetch(接口地址, { method: "GET" })` 发送请求；
   - `.then(response => response.json())` 解析JSON响应；
   - `.then(data => { /* 用DOM显示data */ })` 处理数据；
2. JSON 方法：
   - `JSON.parse()`：把后端返回的JSON字符串转成JS对象（Fetch会自动做，了解即可）；
   - `JSON.stringify()`：把前端数据转成JSON字符串（提交数据时用，如`body: JSON.stringify(newData)`）。

### 入门资源
- 官网教程（免费）：[MDN Fetch教程](https://developer.mozilla.org/zh-CN/docs/Web/API/Fetch_API/Using_Fetch)（看“基本用法”和“发送GET/POST请求”）

### 实操小案例
**案例：用Fetch获取模拟接口数据，插入表格**  
（用[JSONPlaceholder](https://jsonplaceholder.typicode.com/)的模拟接口，不用等后端开发）
```javascript
const queryBtn = document.getElementById("queryBtn");
const dataTable = document.getElementById("dataTable");

queryBtn.addEventListener("click", () => {
  // 发送GET请求（模拟调用后端接口）
  fetch("https://jsonplaceholder.typicode.com/users") // 模拟用户数据接口
    .then(response => response.json()) // 解析JSON
    .then(users => { // users是接口返回的数组数据
      const tbody = dataTable.querySelector("tbody") || dataTable;
      tbody.innerHTML = "";

      // 插入数据（只显示id、name、email，对应数据库字段）
      users.forEach(user => {
        const tr = document.createElement("tr");
        tr.innerHTML = `
          <td>${user.id}</td>
          <td>${user.name}</td>
          <td>${user.email}</td>
        `;
        tbody.appendChild(tr);
      });
    });
});
```


## 五、可视化插件（可选，1天，提升课程设计质感）
### 核心重点（以ECharts为例，中文友好）
1. 引入ECharts：在HTML头部加`<script src="https://cdn.jsdelivr.net/npm/echarts@5.4.3/dist/echarts.min.js"></script>`；
2. 准备容器：用`<div>`设固定宽高（如`#chart { width: 600px; height: 400px; }`）；
3. 基础配置：初始化图表、设置x轴/y轴数据、绑定数据库返回的数据。

### 入门资源
- 官网教程：[ECharts 5分钟上手](https://echarts.apache.org/handbook/zh/get-started/)

### 实操小案例
**案例：用学生成绩数据做柱状图**  
```javascript
// 1. 初始化图表
const chartDom = document.getElementById("chart");
const myChart = echarts.init(chartDom);

// 2. 点击查询按钮，同时渲染表格和图表
queryBtn.addEventListener("click", () => {
  fetch("https://jsonplaceholder.typicode.com/users")
    .then(res => res.json())
    .then(users => {
      // （先做表格插入，代码同上）

      // 3. 处理图表数据（提取name作为x轴，id作为y轴，模拟成绩）
      const xData = users.map(user => user.name); // x轴：姓名
      const yData = users.map(user => user.id * 5); // y轴：模拟成绩

      // 4. 配置图表
      const option = {
        title: { text: "学生成绩分布" },
        xAxis: { type: "category", data: xData },
        yAxis: { type: "value" },
        series: [{ type: "bar", data: yData }] // 柱状图
      };

      myChart.setOption(option); // 渲染图表
    });
});
```
