# 数据挖掘核心知识点总结
## 一、数据挖掘定义
从现有的大量数据中，抽取不明显、之前未知且可能有用的信息。

## 二、数据挖掘相关流程环节
1. 数据来源：包含原始数据等各类相关数据
2. 数据处理环节：
    - 数据选择
    - 数据集成
    - 数据准备
    - 数据预处理
3. 核心环节：数据挖掘（基于处理后的数据开展）
4. 输出结果：模式、知识（用于支持决策等场景）

## 三、数据挖掘关键技术/方法
- 决策树
- 关联规则
- 神经网络

## 四、数据处理相关发展历程
|时间阶段|核心特征|关键技术/载体|
| ---- | ---- | ---- |
|1960s|资料搜集|磁带、软盘、硬盘|
|1980s|数据查询|数据库（SQL语言）|
|1990s|资料统计|资料仓储（OLAP）|
|2011年|数据分析|数据挖掘（Data Mining）|
